Just Another Boss Rush

Just Another Boss Rush was a game made in Godot for my game development class. This project was my first time working in Godot. Just as the name implies this is a boss rush style game where the player fights a series of bosses in order to increase high scores.

State Machine

The state machine was an interesting challenge since it was the part of the game that had to work with other scenes/levels and switch between each one when the player has defeated a boss. This was especially the case when it came to restarting the game when the player beat it because we needed a way to delete the previous scores, times, and level scenes. At first, I thought I could just set the current scene back to the first scene in the scene history, but we would run into an issue where the game levels would appear in the correct sequence but, some transitions would be out of order. Also, the score and time in each full playthrough increased from when it was last played. To fix this bug, I instead cleared the scene history and loaded the levels the same way they were loaded at the start of the game.
